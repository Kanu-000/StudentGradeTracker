# Student Grade Tracker 🎓

A simple and elegant web application to add student grades, calculate averages, and view a detailed performance summary — built as part of my **CodeAlpha Internship** (Web Development track).

## ✨ Features

- **Add Students**: Enter a student's name and comma-separated grades (0–100)
- **Automatic Calculation**: Computes average, highest, and lowest score for each student
- **Letter Grading**: Assigns a grade (A+, A, B, C, D) based on average score
- **Edit & Remove**: Update or delete any student entry
- **Progress Tracker**: Visual dots show how many students (up to 5) have been added
- **Summary Report**: Once 3+ students are added, view a full comparison page with:
  - Top performer highlight
  - Class statistics (total students, class average, highest/lowest score)
  - Visual grade comparison bars
  - Detailed ranked results table

## 🛠️ Tech Stack

- **HTML5** – Structure
- **CSS3** – Styling, gradients, responsive layout, animations
- **Vanilla JavaScript** – Logic, DOM manipulation, dynamic rendering

## 🚀 Running the Project

1. Download/clone the repository
2. Open `index.html` in any web browser
3. Start adding students and view your summary report
