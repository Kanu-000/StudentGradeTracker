Student Grade Tracker
A web-based Student Grade Tracker built during my internship at CodeAlpha. This application allows users to add student grades, automatically calculate the average score, and determine the corresponding letter grade.
Features
Add student names and subject grades
Automatic calculation of average marks
Grade evaluation based on average (A, B, C, etc.)
Clean and responsive UI built with HTML, CSS, and JavaScript
Tech Stack
HTML – Structure
CSS – Styling
JavaScript – Logic and DOM manipulation
